SERIAL OUTPUT COMMAND LIST - 1.0.2

Numerical commands:

COMMAND: POWER_LEVEL
DESCRIPTION: Total current power stored in kw.

COMMAND: POWER_LEVEL_PERCENT
DESCRIPTION: Total current power as a percentage.

COMMAND: POWER_FLOW
DESCRIPTION: Total power flow, in positive or negative amounts.

COMMAND: POWER_FLOW_PERCENT
DESCRIPTION: Total power flow, in percent.

COMMAND: POWER_GENERATION
DESCRIPTION: Total power generated currently.

COMMAND: POWER_GENERATION_PERCENT
DESCRIPTION: Total power generated currently, in percent of theoretical max.

COMMAND: POWER_DRAIN
DESCRIPTION: Total power drain currently.

COMMAND: POWER_DRAIN_PERCENT
DESCRIPTION: Total power drain currently, in percent of theoretical max.

COMMAND: DESIRED_DIRECTION
DESCRIPTION: Desired directions elected on helm panel.

COMMAND: MOTION_ANGLE
DESCRIPTION: Current motion angle.

COMMAND: DIRECTION
DESCRIPTION: Current ship direction.

COMMAND: CURRENT_SPEED
DESCRIPTION: Current speed.

COMMAND: CURRENT_SPEED_PERCENT
DESCRIPTION: Current speed as percent of theoretical max.

COMMAND: PRESSURE_EXTERIOR
DESCRIPTION: Exterior airlock pressure.

COMMAND: PRESSURE_INTERIOR
DESCRIPTION: Interior airlock pressure.

COMMAND: PRESSURE_AIRLOCK
DESCRIPTION: Pressure inside the airlock.

COMMAND: CURRENT_TEMPERATURE
DESCRIPTION: Current temperature inside the ship.

COMMAND: CURRENT_DISTANCE
DESCRIPTION: Current distance to selected object.

COMMAND: CURRENT_BEARING
DESCRIPTION: Current bearing to selected object.

COMMAND: ASTEROID_DENSITY
DESCRIPTION: Current density of asteroid field outside.

COMMAND: NEBULA_DENSITY
DESCRIPTION: Current density of nebula outside.

COMMAND: TUBE_SELECTED
DESCRIPTION: Currently selected tube (0-7).

COMMAND: AMOUNT_OWED_TO_DOCKED
DESCRIPTION: Amount owed to currently docked station.

COMMAND: AMOUNT_OWED_VIA_COMMS
DESCRIPTION: Amount owed to space station currently communicating with.

COMMAND: AUTO_REPAIR_PERCENT
DESCRIPTION: [no longer used]

COMMAND: CURRENT_TUBE_SPIN_UP_PERCENT
DESCRIPTION: Spin up percent of currently selected tube.

COMMAND: TUBE_SPIN_UP_PERCENT
DESCRIPTION: Spin up percent of any tube (parameter: 1-8)

COMMAND: ENG_CURRENT_REPAIR_PERCENT
DESCRIPTION: [no longer used]

COMMAND: JMP_SOLUTION
DESCRIPTION: Jump drive solution percent.

COMMAND: MAIN_DRIVE_POWER_LEVEL
DESCRIPTION: Main drive current power level in percent.

COMMAND: HULL_TEMPERATURE
DESCRIPTION: Current hull temperature.

COMMAND: SOLAR_RADIATION_PERCENT
DESCRIPTION: External solar radiation percent.

COMMAND: GRAPPLE_ARM_PERCENTAGE
DESCRIPTION: Percentage of current grappling arm operation.

COMMAND: HACK_PERCENTAGE
DESCRIPTION: Percentage of current hack operation.

COMMAND: PENDING_EMAILS
DESCRIPTION: Number of emails pending to download.

COMMAND: UNREAD_EMAILS
DESCRIPTION: Number of downloaded but unread emails.

COMMAND: READ_EMAILS
DESCRIPTION: Number of downloaded but read emails.

COMMAND: PLAYER_CREDITS
DESCRIPTION: Current number of player credits.
